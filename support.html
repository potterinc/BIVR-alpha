User Flow

Once the App loads users will enter a login scene

Login Scene: In the Login scene A user will choose to either login anonymouslyor with an ID with with other users can ldentify you. Once loggedin you will enter the.

Home Scene: In the home scene there is an Avatar selector and UI setup forjoining Rooms/Worlds. We currently have three worlds in the BIVRalpha. the Island Boat,Island Sky and Game room.

Island Boat: Starts on a boat sailing towards an island, once the boat docks auser can exit onto the Island(this scene is mainly to show the boatcruise capability)

Island Sky: This is the main world in the BIVR alpha,it is the same island onthe island boat but it is better configured for exploration, it startup in the sky on the wing of an aircraft the user jumps off for askydiving experience onto the island to explore all the different attractions.

Game Room: This room is aprivate room with maximum occupancy of twousers, once a user joins the room with a friend they can voice chatwhile playing the the different games and props.
